# Monorepo – Frontend & Backend

Dự án này được tổ chức theo dạng **monorepo**, bao gồm cả hai phần:

- **frontend/** – mã nguồn giao diện người dùng  
- **backend/** – mã nguồn API & dịch vụ phía server  

Mỗi phần đều có **README.md riêng** để mô tả chi tiết cách cài đặt, chạy và phát triển.

## Cấu trúc thư mục

```
├── frontend/
│ └── README.md # Hướng dẫn sử dụng & phát triển Frontend
├── backend/
│ └── README.md # Hướng dẫn sử dụng & phát triển Backend
└── README.md # File hiện tại (ở root)
```

- Để xem hướng dẫn Backend → vào thư mục **backend** và đọc file `README.md`.
- Để xem hướng dẫn Frontend → vào thư mục **frontend** và đọc file `README.md`.

File README này chỉ dùng để mô tả tổng quan về cấu trúc monorepo.
